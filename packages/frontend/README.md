# @task-assistant/frontend

Next.js application for the Task Assistant UI.

## Tech Stack
- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: React

## Environment
The frontend runs on port `3000` in the Docker container.
It communicates with the backend via `http://localhost:3001` (browser) or internal Docker networking (SSR).

## Scripts
- `dev`: Run Next.js dev server locally
- `build`: Build for production
- `start`: Start production server
